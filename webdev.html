Web Development

Web development is the process of building and maintaining websites. It involves creating the structure, design, and functionality of a website, ensuring it is user-friendly, responsive, and accessible. Web development plays a key role in creating engaging and interactive online experiences for users.

Order Management System

The Order Management System is a special application for Tender. It contains the facilities of Dashboard, Order, Accepted price list, Send request, Check inserting and Trash.

  • Front End - using ReactJs, Bootstrap, Css, HTML5
  • Back End - using JS, Firebase


Edu Connect

EDUconnect is an educational web application. User can follow any course and track their progress. A student can join the community and discuss some issues with other students. This application contains a notification device and the user can change the settings.

  • Front End - using ReactJs, Bootstrap, Css, HTML5, JS
  • Back End - using PHP, SQL and Firebase


Social Media

This social media web application is very similar to other social media, with registration, login pages and functions like uploading posts, liking, commenting, and searching.

  • Front End - using Bootstrap, Css, HTML5, JS
  • Back End - Django
